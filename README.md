# Вычисление пола ребенка

This is a simple Bootstrap template with responsive Navbar.
This for educational use.

## Некоторые формулы для вычисления пола ребенка

 
А+Б
где 
А – возраст матери на момент зачатия,
Б – порядковый номер месяца зачатия.
Если выходит четное число – будет девочка, нечетное число – будет мальчик.
 
(А+Б):3
где А – возраст матери на момент зачатия,
       Б – возраст отца на момент зачатия.
Если в остатке получается 1 – будет девочка,
                                       0 или 2 – будет мальчик.
 
49+А-Б+19
где А – порядковый номер месяца зачатия,
       Б – возраст матери на момент зачатия.
Если выходит четное число – будет девочка,
                      нечетное число – будет мальчик.
 
49-(А(Б+1))
где А – порядковый номер месяца зачатия,
       Б – возраст матери на момент зачатия.
Если выходит четное число – будет девочка,
                      нечетное число – будет мальчик.
